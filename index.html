Pokémon Blue

Versão para jogar direto no navegador (GitHub Pages)

Se a tela estiver preta, clique em Jogar e espere alguns segundos para o emulador carregar a ROM.

Controles

  • Setas – Direcionais
  • Z – Botão A
  • X – Botão B
  • Enter – Start
  • Backspace – Select

Sobre

Este projeto foi preparado para rodar em um site estático (como o GitHub Pages), usando um emulador de Game Boy em JavaScript (WasmBoy).
